# Guardian Doctrine Map

## Purpose

A consolidated map of Guardian's constitutional governance
doctrine, derived from spec development and external
corridor validation.

Date: 2026-05-18

---

## Core Doctrine Layers

### V0.2 — Decision Equivalence
The structural condition under which decisions produced
by independently governed systems remain formally
comparable across governance boundaries.

Key question: Are the decisions the same?

### V0.3 — Acceptance Compatibility
The structural condition under which a decision produced
in one governed system is valid and actionable within
another governed system's authority context.

Key question: Is this decision valid here?

### Section 10 — Anti-Normalization Boundary
Constitutional prohibition against allowing recursive
interoperability to substitute for explicit sovereign
re-resolution.

Key question: Is continuity replacing legitimacy?

### V0.4 Candidate — Sovereign Re-Resolution Capacity
The structural requirement that independently governed
systems preserve the capacity for explicit legitimacy
re-determination under persistent coordination pressure.

Key question: Can each system still independently
resolve legitimacy without assuming continuity?
